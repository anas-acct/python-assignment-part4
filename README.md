# Student Performance Analysis & Prediction

This project is part of my assignment where I worked on analysing student performance data and building a simple machine learning model to predict whether a student will pass or fail.

## 💻 Environment Used
I have completed this project using **Google Colab**, which is an online Jupyter Notebook environment. It allowed me to write and run Python code easily without setting up everything on my local system.

## 📊 About the Dataset
The dataset contains details of 15 students, including:
- Marks in subjects (Math, Science, English, History, PE)
- Attendance percentage
- Study hours per day
- Pass/Fail status

## 🔍 What I Did

### 1. Data Exploration
First, I loaded the dataset using pandas and explored it by:
- Checking the first few rows
- Looking at data types and structure
- Finding summary statistics like mean and standard deviation
- Counting how many students passed and failed
- Comparing average marks of pass and fail students
- Finding the student with the highest average score

### 2. Data Visualization (Matplotlib)
I created different types of plots to understand the data better:
- Bar chart for average subject scores
- Histogram for math marks
- Scatter plot for study hours vs performance
- Box plot for attendance comparison
- Line plot for math and science scores

### 3. Data Visualization (Seaborn)
Using seaborn, I made:
- Bar plots comparing math and science scores for pass/fail students
- Scatter plot with regression lines to see trends between attendance and scores

I found seaborn easier for styling and cleaner
